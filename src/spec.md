# Specification

## Summary
**Goal:** Refine button styling and add support for multiple payment modes per invoice and multiple invoices per payment.

**Planned changes:**
- Make Clear Records and Download Excel buttons smaller and more compact in the PaymentReportTable
- Change Invoice Number field to accept alphanumeric text (e.g., "INV-001", "BILL123")
- Enable selection of multiple payment modes (NEFT and/or Cheque) for a single invoice, showing all relevant fields simultaneously
- Allow entry of multiple invoice numbers for a single payment transaction

**User-visible outcome:** Users can enter invoice numbers with text prefixes, apply one payment across multiple bills, or use multiple payment methods for a single invoice. Action buttons are more compact and streamlined.
